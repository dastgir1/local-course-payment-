# Course Carousel - Moodle Local Plugin

## Description
**Course Carousel** (`local_course_carousel`) is a local plugin for Moodle that provides functionality to display course information, a dedicated signup form, and a payment integration flow. 

## Features
- **Course Information:** Displays detailed information for courses.
- **Custom Signup Form:** Includes a modern, customized signup form (`signupform.php`) to register users directly for specific courses.
- **Payment Integration:** Built-in payment workflow (`payment.php`) for course enrollment.

## Requirements
- Moodle version: 5.0 or 5.1 (Requires build `2025041400` or higher).

## Installation
1. Clone or download this repository.
2. Place the contents into the `local/course_carousel` directory of your Moodle installation.
3. Log in to your Moodle site as an administrator.
4. Go to **Site administration -> Notifications** to complete the plugin installation.
